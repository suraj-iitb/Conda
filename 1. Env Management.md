# Conda Environment Management

## Show All Conda Environments
  `$ conda info --envs`
  
## Create Conda Environment
  `$ conda create -n <env>` 
  
## Remove Conda Environment
  `$ conda remove -n <env> --all`
  
## Activate Conda Environment
  `$ conda activate <env>`
  
## Deactivate Conda Environment
  `$ conda deactivate`
  
## Show All Packages
  `$ conda list`
  
## Install Packages
  `$ conda install <pkg>`
  
## Remove Packages
  `$ conda remove <pkg>`
  
## Update Packages
  `$ conda update <pkg>`
  
## Search Packages
  `$ conda search <pkg>`
