# Conda Environment Management

## Create Conda Environment
  `$ conda create -n <env>` 
  
## Activate Conda Environment
  `$ conda activate <env>`
  
## Deactivate Conda Environment
  `$ conda deactivate`
  
## Install Packages
  `$ conda install <pkg>`
  
## Remove Packages
  `$ conda remove <pkg>`
  
## Update Packages
  `$ conda update <pkg>`
  
## Search Packages
  `$ conda search <pkg>`
  
## Show Packages
  `$ conda list`
